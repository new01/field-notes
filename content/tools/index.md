# Tools

Pipelines and systems built for real automation work. All open-source.

---

## What We've Built

- **OpenClaw skill library** — a growing collection of composable skills for the OpenClaw runtime, covering research, content, infrastructure, and more
- **Whisper transcription pipeline** — local audio-to-text using OpenAI Whisper, running fully on-device with no data leaving the machine
- **Ops dashboard** — an internal dashboard for monitoring agent runs, costs, and system health across all active pipelines
- **Humanization pipeline** — a post-processing pass that strips LLM patterns from generated text, making output read as natural human writing

---

All tooling lives on [GitHub](https://github.com/cyne-wulf). Issues and contributions welcome.
